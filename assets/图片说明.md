# 参考图片命名说明

把图片放进本文件夹 `ui设计图/images/`，Demo 会从这里加载。

## 必需文件

| 文件名 | 用途 |
|--------|------|
| `party-wine-cellar.png` | 酒局邀请卡 · 今晚酒局 |
| `cocktail-amber-winter.png` | 调酒方案 · 琥珀暖冬 |
| `bar-whisky.png` | 附近酒馆 · 琥珀威士忌吧 店内 |
| `bar-bistro.png` | 附近酒馆 · 半酌小馆 店内 |
| `bar-lounge.jpg` | 附近酒馆 · 金夜酒廊 店内 |
| `sample-whisky.png` | 首页示例 · 威士忌（可选） |
| `sample-red.png` | 首页示例 · 红酒（可选） |
| `sample-baijiu.png` | 首页示例 · 白酒（可选） |
| `bartender.png` | 首页老金插画（可选） |

## 启动方式（重要）

必须从 **`金灵酒鬼` 根目录** 启动静态服务，否则图片路径会 404：

```bash
cd 金灵酒鬼
npx serve .
```

浏览器打开：`http://localhost:端口/demo/`

不要只在 `demo` 子目录里 `npx serve`，那样读不到 `ui设计图/images/`。
